# GPI Case

## 资料

* https://support.retroflag.com/
* https://github.com/RetroFlag/retroflag-picase
* https://www.thecoverproject.net/
* https://gamesdb.launchbox-app.com/
* root | recalboxroot
* 退出游戏：HOME+B
* 设置
  * 系统设置-语言-简体中文
  * 用户界面设置
    * GAME FILTERS-隐藏预装游戏
    * 更新游戏列表
  * 网络设置
    * 启用无线网络
    * 无线网络 SSID
    * 无线网络密码
      * START 确认
* 使用蓝牙手柄
  * 进入游戏或者模拟器菜单后，需要按L2+R2才能激活手柄
* NEOGEO 模拟器游戏
  * 游戏 rom 名称不能修改
  * 需要将 neogeo.zip 放在对应 rom 文件夹中
  * 需要将 pgm.zip 放在对应 rom 文件夹中
